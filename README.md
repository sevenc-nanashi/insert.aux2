「メディアを挿入」編集メニューを追加するAviUtl2の汎用プラグイン。
Reaperの<kbd>Insert</kbd>キーみたいなことをAviUtl2でやりたい人向け。

## インストール

[Releases](https://github.com/sevenc-nanashi/insert.aux2/releases/latest)から`insert-v{{version}}.au2pkg.zip`をダウンロードし、AviUtl2のプレビューにドラッグ&ドロップしてください。

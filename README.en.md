# insert.aux2

[English](README.en.md) | [日本語](README.md)

[![AviUtl2 Catalog](https://aviutl2-catalog-badge.sevenc7c.workers.dev/badge/v/sevenc-nanashi.insert-aux2)](https://aviutl2-catalog-badge.sevenc7c.workers.dev/package/sevenc-nanashi.insert-aux2)

AviUtl2 plugin that adds "Insert Media" editing menu.
For those who want to do something like Reaper's <kbd>Insert</kbd> key in AviUtl2.

## Installation

Download `insert-v{{version}}.au2pkg.zip` from [Releases](https://github.com/sevenc-nanashi/insert.aux2/releases/latest), and drag-and-drop it into the AviUtl2 preview.
